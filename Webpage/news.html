Corina
The Cornell Tree-Ring Analysis System

News

Updates

9 Jan 2004

More updates. We're becoming almost stable.

  • Everything in CVS is up-to-date now. You can build the shipping Corina.jar directly from CVS.
  • Lots of bug fixes.
  • Lots of performance enhancements, especially in the sample browser.
  • Much simpler installation (just download and double-click Corina.jar, if you have Java 1.4 or newer).

Web page updated

21 Jan 2003

After many long months with no updates, I've finally started putting together a better web page. Hopefully the information should be easier to find, and more up-to-date.

Mailing Lists

If you want to be informed when something is updated, Corina has 2 mailing lists for users:

  • corina-announce, a low-volume list for announcements about Corina
  • corina-users, a higher-volume list for general discussion
