Installing Corina

Check System Requirements

You'll want a computer with at least a 500 MHz or so processor. If it's slower than that, Corina will still run, but will run slowly.

You can use use any system that Java 1.3/1.4 runs on, including:

  • Mac OS X, 10.1 or newer
  • Linux
  • Windows 98, ME, 2000, or XP

Any system with Java 2 can run Corina, but for best performance, I recommend the following:

  • Java 2, version 1.3.1 or 1.4.2 -- takes ??? hard disk space
  • 500 MHz CPU
  • 128 MB RAM

Install Java (if necessary)

You'll need Java 1.3.1 or 1.4.2 or newer. If you have a Mac, you already have Java installed.

  • Java 1.4.2 "JRE" (you don't need the "SDK").
  • Java 1.3.1 "JRE" (you don't need the "SDK"), OR

Try 1.4.2 first. Only use 1.3.1 if you have problems with 1.4.2.

Install Corina

  1. Download this file: Corina.jar.
  2. Double-click it.
  3. There is no step 3.
