Features

Crossdating algorithms

  • T-Score
  • R-Value
  • Trend (Gleichläufigkeit)
  • D-Score
  • Weiserjahre

Indexing algorithms

  • Exponential
  • Polynomial (any degree)
  • Cubic spline
  • High-pass filter

File formats

  • Corina (native)
  • Tucson (raw or indexed/summed)
  • Hohenheim
  • Heidelberg
  • Two-column (text)
  • TSAP

Other Features

  • Crossdating tables and grids
  • Sample browser
  • Atlas

Miscellaneous

  • OSI Certified Open Source Software
  • Runs on Linux, Mac, and Windows (written in Java)

No measuring stations are natively supported yet (though with information about the hardware, that wouldn't be hard to add).

Corina was born at the Aegean Dendrochronology Project in Ithaca, NY.
