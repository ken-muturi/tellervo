Corina
The Cornell Tree-Ring Analysis System

What is this?

Corina is a program for dendrochronologists. It allows you to:

  • Enter or import data
  • Edit and manipulate data (including indexing)
  • Crossdate samples (using several algorithms)
  • Graph data
  • Keep track of samples and sites
